### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

### Реализован мультипликативный конгруэнтный генератор
```
class MCG:
    def __init__(self, seed):
        self.M = 2**63
        self.beta = 2**32 + 3
        self.x = seed  

    def next(self):
        self.x = (self.beta * self.x) % self.M
        return self.x / self.M
```
### Параметры

1. M = 2^63 - модуль, максимальное состояние генератора
2. beta = 2^32 + 3 - множитель, коэффициент на который умножается текущее состояние
3. seed - начальное состояние генератора

### Вычисление выборочного среднего и дисперсии генераторов на выборке N = 100000
| Характеристика | Теория | MCG | `random.random()` |
|---|---|---|---|
| Среднее | 0.50000 | 0.49972 | 0.49874 |
| Дисперсия | 0.08333 | 0.08310 | 0.08364 |

результаты достаточно близки к теоретическим

### Анализ требований к базовым датчикам
1. Равномерное распределение

Равномерность мультипликативного конгруэнтного генератора обусловлена тем, что операция умножения по модулю при правильно выбранных параметрах задаёт перестановку множества остатков, обеспечивая равномерное покрытие диапазона значений. После нормировки на интервал [0,1) получаем последовательность, приближённо равномерно распределённую, что подтверждается экспериментально

```
MCG:
Среднее:  0.49972619574032257
Дисперсия:  0.08310359482594133
```

2. Отстутстиве корреляции

```
def autocorrelation(arr, lag=1):
    n = len(arr)
    m = mean(arr)

    num = sum((arr[i] - m) * (arr[i + lag] - m) for i in range(n - lag))
    den = sum((x - m) ** 2 for x in arr)

    return num / den
```
Автокорреляция:  -0.00045692744308564086

3. Апериодичность

Модуль = M = 2^63 во много раз больше выборки, поэтому обеспечивается апериодчиность.

4. Воспроизводимость

Генератор использует seed как начальное состояние, поэтому при одинаковом значении этого параметра последовательности будут одинаковы.
